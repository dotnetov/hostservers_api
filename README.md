# Server Hosting API

## Описание проекта

Server Hosting API — это REST API, разработанное на Django REST framework для предметной области **хостинга серверов**.

## Предметная область

Предметная область проекта – **хостинг серверов**.

Система может использоваться для хранения и обработки информации о:

- тарифах хостинга;
- клиентах;
- арендованных серверах/VPS;
- обращениях клиентов в техническую поддержку.


## Основной функционал

В проекте реализованы следующие возможности:

- создание, просмотр, изменение и удаление тарифов хостинга;
- создание, просмотр, изменение и удаление клиентов;
- создание, просмотр, изменение и удаление серверов/VPS;
- создание, просмотр, изменение и удаление обращений в техническую поддержку;
- фильтрация данных через GET-параметры;
- автоматическая генерация Swagger-документации;
- работа с базой данных PostgreSQL;
- управление данными через встроенную Django admin-панель.


## Адреса проекта

| Адрес | Назначение |
|---|---|
| `http://127.0.0.1:8000/api/v1/` | Главная страница API |
| `http://127.0.0.1:8000/admin/` | Админ-панель Django |
| `http://127.0.0.1:8000/api/schema/` | OpenAPI-схема |
| `http://127.0.0.1:8000/api/schema/swagger-ui/` | Swagger-документация |

## Список эндпоинтов

## Tariffs

Эндпоинты для работы с тарифами хостинга.

| Метод | Эндпоинт | Описание |
|---|---|---|
| GET | `/api/v1/tariffs/` | Получить список тарифов |
| POST | `/api/v1/tariffs/` | Создать новый тариф |
| GET | `/api/v1/tariffs/{id}/` | Получить тариф по id |
| PUT | `/api/v1/tariffs/{id}/` | Полностью обновить тариф |
| PATCH | `/api/v1/tariffs/{id}/` | Частично обновить тариф |
| DELETE | `/api/v1/tariffs/{id}/` | Удалить тариф |

### Фильтрация тарифов

| Параметр | Пример | Описание |
|---|---|---|
| `name` | `/api/v1/tariffs/?name=VPS` | Поиск тарифа по названию |
